# **App Name**: FoodVrse Ventures

## Core Features:

- Company Information: Provide comprehensive information about FoodVrse, its mission, and values.
- Portfolio Display: Showcase FoodVerse's investments using imagery and descriptions of the respective companies.
- Impact Metrics: Visually represent FoodVrse's impact through key metrics.
- News and Updates: Provide an 'Updates' section showing 3 milestones or news highlights
- Sub-brand Redirection: Ensure that clicks to sub-brand sites, like 'Dishii', will direct the user to that page on a separate tab.

## Style Guidelines:

- Primary color: Deep Charcoal (#1A1A1A) for headers and logo.
- Secondary color: Warm Gray (#4A4A4A) for body text.
- Background color: Warm Off-White (#FAF7F2) for the entire site background.
- Accent color: Harvest Gold (#B78C5A) for buttons, lines, and the small dot before links.
- Headers: 'Literata' (serif) for a premium, established feel. Note: currently only Google Fonts are supported.
- Body: 'Inter' (sans-serif) for a clean and readable design. Note: currently only Google Fonts are supported.
- Utilize generous letter spacing and line height for an elegant feel.
- Maintain generous margins and padding throughout for negative space.
